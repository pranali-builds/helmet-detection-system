# helmet-detection-system
AI based helmet detection system using Python
## Project Description

This project is an AI-based system that detects helmets, vehicles, and number plates from road images.

Features:
- Helmet detection (Helmet / No Helmet)
- Rider and bike detection
- Vehicle detection
- Number plate recognition using OCR

The system allows the user to upload an image and performs end-to-end analysis using computer vision and deep learning techniques.

## Example Outputs

Below are some example outputs from the system:

1. Bike Only Detection – Detects bike without rider and extracts number plate.
2. No Helmet Detection – Detects riders without helmets and shows confidence score.
3. Helmet Detection – Detects riders wearing helmets correctly.
